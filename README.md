# Monitoramento

É uma parte crucial para garantir

- Performance
- Disponibilidade
- Segurança
- Detençao

Através da coleta e análise de Métricas e Logs!

Criar Sistema de Alerta de tiver um erro no sistema

Sem Monitoramento o sistema pode ficar lento ou mesmo indisponivel!

## Prometheus

Armazena métrica de forma escalável - foi fundada pela SoundCloud em 2012!

Usa linguagem própria para consulta de dados - PromQL

- Sistema Aberto de Monitoramento
- Integração com Diversos Sistemas
- Modelo de Dados (Multidimensional)

## Componentes do Prometheus

Prometheus Server

- Retrieval (Possibilita pegar informações dos target e colocar no TSDB)
- TSDB (Banco de dados de Serviços Temporários)
- HTTP Server (Servidor HTTP)

### Node

- Storage

## Porque usar o Prometheus

Integração com outras ferramentas como o Grafana etc.

- Target
- Exportadores
- Métricas Costumizadas

## Grafana

Auxilia o Prometheus a visualizar dados de forma mais intuitiva!

Cria Dashboard que faciliitam recolher dados
